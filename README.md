# Food Ordering Website

A full-stack food ordering web application built with React.js, Laravel, and PostgreSQL. The application provides a seamless food ordering experience with secure authentication, QR Code-based order validation, and email verification features.

## Live Demo

🔗 Demo Website: https://griya-dhahar-suroboyo.vercel.app/

## Features

* User Authentication (Login & Register)
* Google OAuth Authentication
* Email Verification
* Password Reset via Email
* Food Menu Browsing
* Shopping Cart Management
* Order Processing
* QR Code-Based Order Validation
* Time-Based QR Code Regeneration
* Protected Routes for Unauthorized Users

## Tech Stack

### Frontend
